# regvue

### Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Lints files

```
npm run lint
```

### Formats files

```
npm run format
```

### Run Playwright Tests

```
npx playwright test
```

Run subset of tests on Chromium only

```
npx playwright test --project chromium --grep html
```
